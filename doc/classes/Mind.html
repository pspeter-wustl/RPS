Class Mind
In: app/models/mind.rb
Parent: Object

The Mind class stores a vector in the three dimensional rock, paper, scissors space. The mind class can also return what value the computer should play

add(count, pattern_length, history_length, move)

Updates the mind use weighted exponential smoothing. The weight is based on the (number of times the pattern was seen)/(length of history / length of the pattern). This can be simplified to (pattern.count * pattern.size) / history.size
